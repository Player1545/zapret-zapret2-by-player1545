## Быстрый старт

1. **Скачайте** архив со страницы [Releases](../../releases).
2. **Разархивируйте** папку в любое удобное место.
3. **Запустите** нужный `.bat` файл от имени администратора.

> **Рекомендация:** используйте `general.bat` или `general(ALT9).bat` — они наиболее стабильны.

Если при запуске появится предупреждение безопасности PowerShell:

```
Security Warning: Script Execution Risk
Invoke-WebRequest parses the content of the web page...
```

Введите `y` или `a` для продолжения.

> **Важно:** программы `winws` и `winws2` могут запросить права администратора — разрешите их запуск. Обе программы имеют открытый исходный код: [zapret](https://github.com/bol-van/zapret), [zapret2](https://github.com/bol-van/zapret2).

## Что такое zapret?

**zapret — это НЕ VPN.**

Он не меняет ваш IP-адрес и не пропускает трафик через иностранные серверы. Это утилита **десинхронизации DPI** (Deep Packet Inspection) на стороне провайдера — она «дурит» систему глубокой проверки пакетов, не затрагивая вашу геолокацию.

## Структура проекта

```
zapret-zapret2-by-player1545
 ├── general.bat              — основная стратегия
 ├── general (ALT...).bat     — запасные стратегии
 ├── service.bat              — инструменты управления (удаление сервиса, игровой фильтр и др.)
 ├── bin/                     — бинарные файлы (winws + winws2)
 ├── lists/                   — листы хостов и IP-адресов
 │   ├── list-general.txt     — основные хосты
 │   ├── ipset-all.txt        — основные IP-диапазоны
 │   ├── list-telegram.txt    — хосты Telegram
 │   ├── ipset-telegram.txt   — IP-диапазоны Telegram
 │   ├── list-google.txt      — хосты YouTube
 │   ├── list-auto.txt        — автодобавление хостов (только для ALT2.bat)
 │   ├── list-exclude.txt     — исключения (DPI не применяется)
 │   ├── ipset-exclude.txt    — исключения по IP (DPI не применяется)
 │   └── ipset-all.txt.backup — резервный ipset с расширенными диапазонами
 └── lua/                     — Lua-скрипты для winws2 (из репозитория zapret2)
```

## Про service.bat

Пункт **1** (установка как сервис) временно отключён: `service.bat` изначально рассчитан только на `winws.exe`, тогда как в этом репозитории используется ещё и `winws2.exe`. Совместная установка пока не поддерживается.

По умолчанию включён **Game Filter**. Если он вам не нужен, отключите его через `service.bat`, введя цифру `6`.

## Неработающие звонки в Discord

Попробуйте решение из [этого issue](https://github.com/Flowseal/zapret-discord-youtube/issues/8028).

Если звонки всё ещё не работают, смените регион голосового сервера на **Rotterdam**:

- **На сервере:** зайдите в настройки голосового канала → «Назначение региона» → выберите `Rotterdam` (находится под `Japan`). Требуются права на редактирование канала.
- **В личных сообщениях:** нажмите «Другие настройки» (`...` рядом с кнопкой «Отключиться») → «Регион» → `Rotterdam`.

Если и это не помогло — рекомендуется **Cloudflare Warp** или **Внезапный Переезд в Нидерланды** 🇳🇱.

## Зачем этот форк?

Форк появился потому, что перенести стратегию из `zapret-discord-youtube` в `winws2` (с фильтрацией UDP-портов для игр) оказалось нетривиальной задачей. Различные параметры `fake` и методы десинхронизации не давали результата — игры просто не загружались.

Главное преимущество `zapret2` / `winws2` — расширенная поддержка протоколов для пейлоадов и L7-фильтрации, ради чего и стоит эта возня.
